# SportsCollects

Marketplace de coleccionables deportivos construido con Vue.js 3 y PHP.

## Instalación

### Backend
1. Importar `database/schema.sql` en MySQL
2. Configurar credenciales en `backend/config.php`
3. Asegurar que XAMPP esté corriendo

### Frontend
1. Navegar a la carpeta frontend
2. Ejecutar `npm install`
3. Configurar Firebase en `src/firebase/index.js`
4. Ejecutar `npm run dev`

## Stack
- Frontend: Vue 3, Vite, Bootstrap 5, Leaflet
- Backend: PHP 8, MySQL
- Auth: Firebase
